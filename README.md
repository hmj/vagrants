# 概要

# 確認した環境

## Virtualbox

バージョン 5.0.12 r104815

## Vagrant

Vagrant 1.7.4

## Box ファイル

http://www.vagrantbox.es/ より Ubuntu14.04のboxを追加しました。

```sh
$ vagrant box add ubuntu1404 https://cloud-images.ubuntu.com/vagrant/trusty/current/trusty-server-cloudimg-amd64-vagrant-disk1.box
```
